Display Options

Tooltips (Hover): Hover over any region to see its name and value in a popup.

Permanent Labels: Display labels directly on the map for all regions.

Info Panel: The upper-right panel shows detailed information when hovering.

Example 1: Tooltips on Hover

Hover over any region to see a popup with the region name and value.

üìù HTML Markup

<philippines-map
    id="map1"
    level="1"
    title="Philippine Regions"
    value-name="Population (millions)"
    color-scheme="blue"
    height="500px">
</philippines-map>

<script>
const map1 = document.getElementById('map1');

// Load data when map is ready
map1.addEventListener('map-ready', () => {
    map1.setData({
        'PH01': 5.3,  // Ilocos Region
        'PH02': 3.7,  // Cagayan Valley
        'PH03': 12.4, // Central Luzon
        'PH04': 16.2, // CALABARZON
        'PH05': 6.1,  // Bicol Region
        'PH06': 7.9,  // Western Visayas
        'PH07': 8.1,  // Central Visayas
        'PH08': 4.5,  // Eastern Visayas
        'PH09': 3.8,  // Zamboanga Peninsula
        'PH10': 5.2,  // Northern Mindanao
        'PH11': 5.6,  // Davao Region
        'PH12': 4.9,  // SOCCSKSARGEN
        'PH13': 13.9, // NCR

// No need to set show-tooltips="true"
</script>
Example 2: Permanent Labels

Labels are always visible on each region.

üìù HTML Markup

<philippines-map
    id="map2"
    level="1"
    title="Philippine Regions with Labels"
    value-name="GDP (billion PHP)"
    color-scheme="green"
    show-labels="true"
    height="500px">
</philippines-map>

<script>
const map2 = document.getElementById('map2');

// Load data when map is ready
map2.addEventListener('map-ready', () => {
    map2.setData({
        'PH01': 432.5,  // Ilocos Region
        'PH02': 286.3,  // Cagayan Valley
        'PH03': 1876.4, // Central Luzon
        'PH04': 2453.7, // CALABARZON
        'PH05': 398.2,  // Bicol Region
        'PH06': 612.8,  // Western Visayas
        'PH07': 743.9,  // Central Visayas
        'PH08': 289.1,  // Eastern Visayas
        'PH09': 234.6,  // Zamboanga Peninsula
        'PH10': 412.3,  // Northern Mindanao
        'PH11': 589.7,  // Davao Region
        'PH12': 387.2,  // SOCCSKSARGEN
        'PH13': 7823.6, // NCR
        'PH14': 156.4,  // CAR
        'PH16': 234.8,  // Caraga
        'PH17': 267.3,  // MIMAROPA
        'PH19': 312.5   // BARMM
    });
});

// Toggle labels programmatically
map2.toggleLabels(true);  // Show labels
map2.toggleLabels(false); // Hide labels
</script>
Example 3: All Provinces (Info Panel Only)

Tooltips disabled - hover information shown only in the upper-right info panel.

üìù HTML Markup

<philippines-map
    id="map3"
    level="2"
    title="All Provinces"
    value-name="Population Density"
    color-scheme="purple"
    show-tooltips="false"
    height="500px">
</philippines-map>

<script>
const map3 = document.getElementById('map3');

// Disable tooltips (only info panel shows data)
// set show-tooltips="false" in HTML

// Or toggle programmatically:
map3.toggleTooltips(false); // Disable tooltips
map3.toggleTooltips(true);  // Enable tooltips

// Combine labels and tooltips:
map3.toggleLabels(true);    // Show permanent labels
map3.toggleTooltips(false); // Disable hover tooltips

// Listen to events
map3.addEventListener('region-click', (event) => {
    const feature = event.detail.feature;
    console.log('Clicked province:', feature.ADM2_EN);
});
</script>
